# hotel-amrai-website
Hotel Amrai Website
A responsive and user-friendly hotel website developed to showcase the services, rooms, and amenities of Hotel Amrai. The project focuses on creating a clean interface with interactive features to enhance user experience.
The website includes multiple pages such as Home, Rooms, Services, and Contact. Users can explore different room options, view hotel facilities, and interact with elements like booking buttons and images. JavaScript is used to add basic interactivity like booking alerts and user feedback messages.

Technologies Used
 HTML
 CSS
 JavaScript

Key Features
 Multi-page website (Home, Rooms, Services, Contact)
 Room details with pricing and amenities
 Interactive buttons (Booking confirmation alerts)
 Clickable images with user feedback
 Clean and simple UI design

Purpose of the Project
This project was created as a beginner-level web development project to practice front-end skills and understand how websites are structured and made interactive.
